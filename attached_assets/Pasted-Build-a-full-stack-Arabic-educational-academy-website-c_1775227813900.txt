Build a full-stack Arabic educational academy website called "نور أكاديمي" (Nour Akademi).

---

## TECH STACK
- Frontend: React + TypeScript + Vite
- Styling: Tailwind CSS + shadcn/ui components
- Routing: React Router v6
- Backend: Express.js (port 5000, serves both API and frontend)
- Database: PostgreSQL with Drizzle ORM
- Language/Direction: Arabic, RTL layout

## DATABASE
PostgreSQL table `students`:
- id SERIAL PRIMARY KEY
- name VARCHAR(255)
- phone VARCHAR(50)
- course VARCHAR(255)
- payment_method VARCHAR(100)
- status VARCHAR(50) DEFAULT 'pending'
- created_at TIMESTAMP DEFAULT NOW()

---

## BACKEND API ENDPOINTS

### POST /api/register
- Validates body with Zod: { name, phone, course, payment_method }
- Inserts student into PostgreSQL
- Fire-and-forget: sends form data (application/x-www-form-urlencoded) to a Google Apps Script URL as: name, phone, course, payment, payment_method
- Returns: { success: true, student: { ...fields } }
- Does NOT block on Google Sheets — log errors but always return 201 if DB insert succeeded

### GET /api/students
- Returns all students ordered by created_at

### GET /api/export
- Fetches all students
- Returns a downloadable CSV file named students_YYYY-MM-DD.csv
- CSV columns: ID, الاسم, الهاتف, الدورة, طريقة الدفع, الحالة, تاريخ التسجيل
- Use withBOM: true so Arabic displays correctly in Excel

### GET /health
- Returns: { status: "ok" }

## SERVER SETUP (server/index.ts)
- Express on port process.env.PORT || 5000, bound to 0.0.0.0
- In development: use Vite as middleware (middlewareMode: true, hmr shared with httpServer, allowedHosts: true)
- In production (NODE_ENV=production): serve static dist/ folder, catch-all serves index.html using app.use() not app.get("*")
- Use createServer from "http" and share with Vite HMR so only ONE port is ever open

---

## DEPLOYMENT
- Build: npm run build (Vite builds frontend to dist/)
- Run: NODE_ENV=production npx tsx server/index.ts
- Target: autoscale
